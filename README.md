# UI/UX Design Patterns Masterguide

Interaktivní katalog osvědčených řešení pro UI/UX design. Vycházíme z knihy Jennifer Tidwell: **Designing Interfaces**.

## Spuštění lokálně

**Předpoklady:** Node.js

1. Nainstaluj závislosti:
   ```bash
   npm install
   ```
2. Spusť aplikaci:
   ```bash
   npm run dev
   ```
3. Otevři http://localhost:3000
